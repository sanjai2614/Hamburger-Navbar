# Responsive Hamburger Navbar

Live Link (https://hamburger-navbar.vercel.app/)

A simple **responsive navigation bar with hamburger menu** built using HTML, CSS, and JavaScript.
The navigation menu automatically switches to a hamburger toggle on smaller screen sizes.

## Features

* Responsive navbar layout
* Hamburger menu for mobile screens
* Smooth show / hide animation
* Glassmorphism blur effect using `backdrop-filter`
* Simple and beginner-friendly JavaScript toggle logic

## Technologies Used

* HTML5
* CSS3 (Flexbox + Media Queries)
* JavaScript (DOM manipulation)
